## 第一步：付款后收邮件，找到你的 VPS 信息

付款成功后，搬瓦工会给你的注册邮箱发一封确认邮件。

**邮件里包含三件宝贝：**

- VPS 的 IP 地址（公网 IP，用来连接服务器）
- SSH 端口号（不是默认的 22，是随机生成的，通常是五位数字）
- 初始 root 密码

这三个信息是你后面登录服务器的钥匙，先找个地方记下来。

没收到邮件怎么办？检查一下垃圾邮件文件夹，一般在那里。如果确实没有，也不用慌，进 KiwiVM 控制面板一样能拿到，下面第二步会说。

## 第二步：登录 KiwiVM 控制面板

KiwiVM 是搬瓦工官方的 VPS 管理后台，所有操作——开机关机、重装系统、切换机房、看 IP——都在这里完成。

**进入方式：**

1. 登录搬瓦工官网后台（客户区）
2. 点击顶部菜单 Services → My Services
3. 找到你购买的 VPS，点击右侧 **"KiwiVM Control Panel"** 按钮

进去之后，左侧有一大堆菜单，先别被吓到。你现在只需要关注首页那块信息面板，里面写着你的 IP 地址和 SSH 端口。

如果界面显示 VPS 状态是 **"Stopped"**，点一下 **"Start"** 开机，等个一两分钟再操作。

## 第三步：重置 root 密码（关键步骤，别跳过）

现在搬瓦工的机制改了——购买后系统不会自动发密码，需要你手动在 KiwiVM 里重置一次。

**操作步骤：**

1. 进入 KiwiVM 控制面板
2. 左侧菜单找到 **"Root password modification"**
3. 点击 **"Generate and set new root password"**
4. 等几秒钟，页面会显示一串新密码

**重要提示：这个密码只会显示这一次。** 记下来，截图也行，总之别关掉页面就忘了。忘了也不是死路一条——再点一次重置就能生成新密码。

另外注意：重置密码需要 VPS 处于**开机**状态。如果你之前关机了，记得先在面板里点 Start 开机，再来重置。

## 第四步：用 SSH 工具连上服务器

现在你手里有了三样东西：IP 地址、SSH 端口、root 密码。可以连了。

用户名固定是 `root`（全小写），不用找。

### Windows 用户

推荐用 **Termius** 或者 **Xshell**，界面友好，按提示填入 IP、端口、用户名和密码就行。

也可以直接用 PowerShell 或者 Windows Terminal，命令格式是：


ssh root@你的IP地址 -p 你的SSH端口号


### macOS / Linux 用户

系统自带终端就够了，打开终端，输入：

bash
ssh root@你的IP地址 -p 你的SSH端口号


回车，输入密码（输入时屏幕不显示字符，这是正常的），回车确认。

出现类似 `root@bwh-xxxxxxx:~#` 的提示符，说明你已经成功登录服务器了。

## 常见问题快答

**买完没有收到邮件怎么办？**
查垃圾箱。如果确实没有，进 KiwiVM 面板查看 IP 和端口，密码手动重置一次。

**SSH 连不上是什么原因？**
先进 KiwiVM 确认 VPS 状态是 "Running"。如果状态正常但连不上，可能是 IP 被墙了——在 KiwiVM 里有官方黑名单检测工具，或者用 tcp.ping.pe 输入你的 IP 和端口测一下，国内全红国外全绿就是 IP 问题，需要付费换 IP 或者切换机房。

**密码输入正确但提示 Permission denied？**
检查用户名是不是小写的 `root`，密码有没有多余空格。还不行就进 KiwiVM 再重置一次密码。

**想换操作系统怎么操作？**
KiwiVM 左侧菜单有 **"Reinstall OS"**，支持多个版本的 Debian、Ubuntu、CentOS 等，选好点确认，几分钟完成。重装之后密码会更新，记得再重置一次。
